# Malaysia Population and GDP Analysis

This project analyzes population distribution across Malaysian states and explores its relationship with state-level Gross Domestic Product (GDP). The study uses data from the Department of Statistics Malaysia (DOSM) covering multiple years.

The analysis aims to identify population concentration patterns and understand whether states with higher populations also tend to have higher economic output. The dataset is processed and visualized using Python (Pandas, Matplotlib, Seaborn) and supported by Spark SQL (Hive Views) for additional data querying.

The results provide insights into regional disparities in population and economic development across Malaysia.
